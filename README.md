# 📱 Lobby — Consumer Mobile App

A consumer mobile application built with React Native, available live on Google Play Store.

## Features

- Cross-platform mobile app (Android / iOS)
- Smooth native-like UI/UX
- Real-time features
- Optimized performance for production

## Tech Stack

- **Framework:** React Native
- **Language:** JavaScript
- **State Management:** Redux / Context API
- **Navigation:** React Navigation
- **Backend:** REST API

## Live on Google Play

[![Google Play](https://img.shields.io/badge/Google_Play-Available-green?logo=google-play)](https://play.google.com/store)

## Setup

```bash
git clone https://github.com/devslotcy/lobby
cd lobby
npm install
npx react-native run-android
# or
npx react-native run-ios
```
